# Kriti: Bangla Fonts CDN & Hosted Fonts

![Kriti Fonts Banner](assets/banner-1544x500.png)

A powerful WordPress plugin designed to seamlessly integrate high-quality Bangla typography into your website. Serve fonts with exceptional speed via the Kriti CDN, or download and host them locally to maintain absolute control over site performance and data privacy.

## Features

* **High Performance**: Serves highly optimized `.woff2` font files for maximum speed.
* **Flexible Delivery**: Choose between our reliable global CDN or dynamic self-hosting on your own server.
* **Targeted Typography**: Apply fonts globally for a unified look, or specifically target semantic elements like headings (H1-H6) and paragraphs.
* **Live Admin Preview**: Preview fonts using your own custom text directly within the WordPress dashboard.
* **Search & Filter**: Discover the perfect Bangla font for your brand from a continuously updated catalog.

## Installation

1. Download the plugin as a `.zip` file from the [Releases](#) tab or clone this repository.
2. Navigate to your WordPress Admin > **Plugins** > **Add New** > **Upload Plugin**.
3. Upload the `.zip` file and click **Install Now**.
4. Activate the plugin.
5. Locate **Kriti Fonts** in the left sidebar dashboard menu.

## Usage

1. Open the **Kriti Fonts** settings page in the WordPress dashboard.
2. Select your preferred **Delivery Method**: `Font CDN` or `Host Font Files`.
3. Browse the typography catalog and click on any font card.
4. In the preview modal, test the font text and choose your assignment scope:
    - **Global**: Applies to everything (body, paragraphs, headings, lists, links).
    - **Custom**: Targets specifically mapped elements.
5. Click **Select & Save Font**. Your chosen typography will instantly reflect on the front end.

## Requirements

* **WordPress**: 5.8 or higher
* **PHP**: 7.4 or higher
