# Kriti: Bangla Fonts CDN & Hosted Fonts

![Kriti Fonts Banner](assets/banner-1544x500.png)

A powerful WordPress plugin to integrate high-quality Bangla typography into your website. Serve fonts lightning-fast via Kriti CDN, or download and host them locally to maintain full control over performance and privacy.

## ✨ Features

- 🚀 **Lightning Fast**: Serves highly optimized `.woff2` font files.
- 🌐 **CDN or Local Hosting**: Choose between our global CDN or self-hosting the fonts on your own server dynamically.
- 🎯 **Targeted Typography**: Apply fonts Globally to affect the whole site, or specifically to Headings (H1-H6) or Paragraphs.
- 👀 **Live Admin Preview**: Test fonts with your own text right inside the WordPress dashboard.
- 🔍 **Search & Filter**: Easily find the perfect Bangla font for your brand from a constantly updated catalogue.

## 📦 Installation

1. Download the plugin as a `.zip` file from the [Releases](#) tab or clone this repository.
2. Go to your WordPress Admin > **Plugins** > **Add New** > **Upload Plugin**.
3. Upload the `.zip` file and click "Install Now".
4. Activate the plugin.
5. Navigate to **Kriti Fonts** in the left sidebar (look for the Bangladesh flag icon 🇧🇩).

## 💻 Usage

1. Go to the **Kriti Fonts** settings page in the dashboard.
2. Select your preferred **Delivery Method** (`Font CDN` or `Host Font Files`).
3. Browse the catalog and click on any font card.
4. In the resulting modal, preview the font text and choose your assignment:
    - **Global**: Changes everything (body, paragraphs, headings, lists, links).
    - **Custom**: Target specifically mapped elements.
5. Click **Select & Save Font**. Your font should instantly reflect on the front end!

## 🧩 Requirements
* **WordPress**: 5.8+
* **PHP**: 7.4+
